# AI-Summarizer

AI-Summarizer is a simple tool that uses artificial intelligence to generate concise summaries from long texts. Just provide any article or document, and the app will automatically create a short summary highlighting the most important points.

## Usage

1. Clone the repository:
    ```bash
    git clone https://github.com/antek042/AI-Summarizer.git
    ```
2. Install dependencies according to your environment.
3. Run the app and input the text you want to summarize.

## Features

- Fast and automatic text summarization
- Easy to use
- Based on modern AI techniques
